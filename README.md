# My Second Brain

This is my personal knowledge base and memory system connected to OpenCode.

## Structure

- **Areas/** - Projects and focus areas
- **People/** - Contact notes
- **Projects/** - Active project notes
- **Journal/** - Daily logs
- **Decisions/** - Architectural and project decisions
- **Memory/** - Cross-session learnings

## Notes

- OpenCode will use `obsidian-second-brain` skill to interact with this vault
- Daily notes go in `Journal/YYYY-MM-DD.md`
- Project context in `Projects/`
- Decisions in `Decisions/`
